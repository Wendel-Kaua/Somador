# Projeto em HTML, CSS e JavaScript

Este projeto foi desenvolvido com o objetivo de praticar e aplicar conhecimentos em HTML5, CSS3 e JavaScript.

Aqui são testados conceitos como manipulação do DOM, eventos, estilização com CSS e lógica básica de programação.

## Objetivo

Colocar em prática o que estou aprendendo no desenvolvimento web, construindo pequenos projetos funcionais.

## Atualizações

O projeto será atualizado constantemente conforme avanço nos estudos, com commits registrando melhorias e novas funcionalidades.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
